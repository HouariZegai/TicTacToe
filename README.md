# TicTacToe 🎮
A simple tic tac toe game app.

## Used language
* Java (Swing).

## How to play
Select type of game:
* With Friend
* With Computer (PC)
  1. Easy
  2. Medium
  3. Hard
  
**PS:** (challange for you), You can not win the computer in the **Hard** case 😀.
